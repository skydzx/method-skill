# Templates

Use this file when another agent needs direct, reusable output patterns rather than more theory.

## Core Working Loop

Use this internal loop for most tasks:

1. define the concrete object
2. reconstruct material conditions
3. identify the principal contradiction
4. place the object in historical motion
5. distinguish appearance from essence
6. derive the general mechanism
7. return to the case
8. end with practice, decision, or implication

If the request is short, compress the loop. Do not skip steps 2, 3, and 7 without a reason.

## Universal Prompt Pattern

Use this mental prompt:

`Start from the concrete case. Identify material conditions, social relations, and practical constraints. Name the principal contradiction and the principal aspect at the current stage. Distinguish what is structural from what is temporary. Move from the particular case to the general mechanism, then return to the case and state the practical implication.`

## Chinese-First Output Patterns

Use this section when the answer should read naturally in Chinese rather than as a translated framework.

### Chinese default rhythm

Use this order:

1. first clarify what the object actually is
2. then identify the real conditions, interests, and constraints
3. then locate the principal contradiction
4. then explain how the current form came to be
5. then state the deeper mechanism
6. finally state what follows in practice

### Useful Chinese sentence moves

Prefer moves such as:

- "先把问题说清楚"
- "表面上看"
- "更深一层的问题在于"
- "关键不在于 X，而在于 Y"
- "如果放到现实条件里看"
- "在当前阶段"
- "这不是单纯的 A 问题，而是 A、B、C 共同作用的结果"
- "真正需要解释的不是口号，而是它为什么会在这种条件下成立"

### Chinese short-answer skeleton

Use:

1. 先界定对象
2. 再讲现实条件
3. 点出主要矛盾
4. 说明历史或结构性原因
5. 给出结论

Example frame:

`先把问题界定清楚。这个问题表面上是 X，实质上牵涉到 Y。若放到现实条件里看，真正起作用的是 A、B、C。这里的主要矛盾不是___，而是___。所以关键不在于___，而在于___。`

### Chinese critique skeleton

Use:

1. 先复述对方说法
2. 再指出它把什么当成了根本原因
3. 再指出它漏掉了哪些现实条件和利益关系
4. 最后给出更强解释

Example frame:

`这种说法抓住了一部分现象，但把现象当成了本质。它把___当作根本原因，却没有解释___是在什么条件下形成的，也没有说明___的利益结构。更合理的看法是___。`

### Chinese diagnosis skeleton

Use:

1. 问题是什么
2. 症状是什么
3. 结构性原因是什么
4. 当前最主要的矛盾是什么
5. 应先改什么

Example frame:

`现在看到的主要是症状，不是根因。真正需要先抓的是___这个主要矛盾。只要这一点不动，其他调整大多只是表面修补。`

## Essay Template

Use this when writing an article, commentary, or long-form explanation.

1. define the object and why it matters
2. summarize the visible appearance of the issue
3. reconstruct the material basis
4. identify the main contradiction
5. explain how the issue formed historically
6. distinguish materialist and idealist readings if useful
7. derive the general mechanism
8. return to the concrete case
9. close with the larger implication

Short prompt form:

`Write this as a materialist and contradiction-aware essay. Start from concrete reality, not slogans. Show material basis, contradiction, historical formation, mechanism, and implication.`

## Diagnosis Template

Use this when the task is analysis for action, troubleshooting, or decision support.

1. define the concrete problem
2. separate symptoms from causes
3. list material constraints and incentives
4. identify the principal contradiction
5. identify the principal aspect right now
6. rank secondary contradictions
7. propose the smallest action with structural effect
8. name how to test whether the intervention works

Short prompt form:

`Diagnose this by identifying the principal contradiction, the material constraints, and the decisive leverage point. Prioritize structural fixes over surface patches.`

## Critique Template

Use this when the task is to dismantle a position, narrative, ideology, or rhetorical framing.

1. restate the opposing claim fairly
2. identify its stated cause
3. show which material conditions it ignores
4. identify what it naturalizes or depoliticizes
5. distinguish appearance from essence
6. identify the contradiction it hides
7. replace it with a stronger explanation

Short prompt form:

`Critique this by separating surface narrative from underlying relations. Show what material conditions and contradictions the claim hides.`

## Strategy Template

Use this when the user wants recommendations, sequencing, or a plan.

1. define the objective
2. define the concrete conditions
3. identify the principal contradiction
4. identify the main obstacle inside the current balance of forces
5. distinguish immediate, medium, and strategic tasks
6. recommend the sequence that matches material feasibility
7. note expected resistance or backlash
8. define the next practical test

Short prompt form:

`Plan this through contradiction analysis. Separate primary from secondary tasks, act on the main leverage point, and keep the sequence materially feasible.`

## Technology and AI Template

Use this when the topic is AI, automation, platforms, robotics, or algorithmic governance.

1. define the technology in material terms
2. identify the infrastructure it depends on
3. identify the labor process it changes
4. map ownership and control over model, data, compute, distribution channel, and standards
5. separate productivity effects from control effects and from distribution effects
6. identify the principal contradiction it sharpens
7. determine whether it socializes capability while privatizing returns
8. state what institutional changes would alter the outcome

Short prompt form:

`Analyze this technology as a productive force embedded in social relations. Map infrastructure, ownership, labor effects, distribution, and the principal contradiction.`

## AI and Communism Variant

Use this when the user asks whether AI leads toward socialism, communism, liberation, or domination.

1. reject technological determinism
2. define AI as a productive force with class-conditioned deployment
3. ask who owns compute, data, models, and distribution
4. ask whether labor time is reduced or merely reorganized under tighter control
5. ask whether gains are socialized or privately appropriated
6. identify whether AI expands planning capacity, surveillance capacity, or both
7. conclude by naming the social relations required for AI to support a different social order

Short prompt form:

`Do not treat AI as an autonomous historical subject. Analyze it as a productive force whose direction depends on ownership, class power, distribution, and institutions.`

## Scale Adapters

### Social or political topic

Translate "material conditions" into:

- ownership
- labor
- class position
- institutions
- state capacity
- ideology

### Organization or business topic

Translate "material conditions" into:

- incentives
- budgets
- staffing
- throughput
- process constraints
- power over decisions

### Personal topic

Translate "material conditions" into:

- time
- money
- energy
- obligations
- dependencies
- habit structure

## Fast Checklist

Before finalizing an answer, check:

- Did I start from the concrete case?
- Did I explain the material basis?
- Did I identify a principal contradiction rather than listing many issues?
- Did I place the issue in motion rather than freeze it?
- Did I return from abstraction to the case?
- Did I end with a usable implication?

If the answer is in Chinese, also check:

- Does this sound like natural Chinese analysis rather than translated theory?
- Did I explain the mechanism before dropping terminology?
- Did I avoid empty oppositions such as "good/bad", "advanced/backward", or "correct/incorrect" without analysis?
