# Chinese Style

Use this file when the output should sound like clear Chinese analysis rather than translated framework language.

## Core Rule

Write in a Chinese analytic rhythm:

- first define the object
- then explain the conditions
- then identify the contradiction
- then state the mechanism
- finally draw the implication

Do not start by stacking terms such as "dialectics", "historical materialism", and "base/superstructure" unless the user explicitly wants that vocabulary up front.

## Sentence Discipline

Prefer:

- short to medium sentences
- one key claim per sentence
- concrete verbs over abstract noun piles
- transitions that show movement: "surface / deeper level / under these conditions / at this stage / therefore"

Avoid:

- long translated chains of abstract nouns
- repeated use of the same formula in every paragraph
- empty emphasis without explanation
- abrupt moral condemnation in place of analysis

## Preferred Chinese Moves

Use patterns like these:

- 先把对象界定清楚。
- 先看现实条件，再看观念表达。
- 表面上是 X，更深一层是 Y。
- 关键不在于 X 本身，而在于它所处的条件和关系。
- 真正需要解释的是：为什么它会在这种条件下出现。
- 如果放回到历史过程里看，问题就不一样了。
- 不能只看立场，还要看结构、利益和约束。
- 真正决定走向的，不只是口号，而是所有权、分配和组织方式。

## Tone Targets

Aim for:

- direct
- sober
- concrete
- layered
- explanatory before evaluative

Do not aim for:

- performative radicalism
- slogan intensity
- excessive certainty without conditions
- decorative theory language

## Chinese Answer Frames

### Short frame

`先把问题界定清楚。表面上看，这是___；但如果放到现实条件里看，真正起作用的是___。这里的关键矛盾不在___，而在___。所以结论是___。`

### Standard frame

`先看对象本身。这个问题不能只从表面现象理解，还要放到现实条件、利益关系和历史过程里看。表面上是___，更深一层则是___。在当前阶段，主要矛盾是___。只要这个矛盾不被触及，其他做法大多只是表面修补。因此，关键不在于___，而在于___。`

### Critique frame

`这种说法并非毫无根据，但它把___当成了根本原因，因此遮蔽了___。如果只停留在这种解释上，就会把结构性问题改写成个人、道德或观念问题。更合理的分析路径是：先看___，再看___，最后再看___。`

### Strategy frame

`如果目的是解决问题，就不能只在表面现象上做文章。当前首先要抓的不是___，而是___这个主要矛盾。因为只要这一点不变，其他调整都很容易被原有结构吸收掉。`

## Fast Repair Moves

If a draft sounds stiff or translated, repair it by:

1. deleting one-third of the theory terms
2. replacing one abstract sentence with one concrete sentence
3. changing "this reflects..." into "this happens because..."
4. changing "from the perspective of..." into "if we look at the actual conditions..."
5. ending with one practical implication
